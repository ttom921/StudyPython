# GopClient

加入`socket.io`

```
npm install socket.io --save
```

加入`PySocketio`的服務

```
ng g s service\PySocketio
```

加入`ChatLoddy`的元件

```
ng g c dashboard\ChatLoddy
```

加入聊天房間

```
ng g c dashboard\ChatRoom
```

加人PySenderLobby

```
ng g c dashboard\PySenderLobby
```

加入讀圖檔的服務

```
ng g s service\ImageFile
```

加入SendBufRoom

```
ng g c dashboard\SendBufRoom
```
